# 🎮 Unity 3D Game Development Portfolio (HDRP)

This repository contains the C# logic for 3D games developed at **Rathmines College**. These projects utilize the **High Definition Render Pipeline (HDRP)** for high-fidelity visuals.

## 🏎️ Featured Projects
### 1. High-Speed Car Racing Game
* **PlayerController.cs**: Manages vehicle physics, including acceleration, steering, and braking.
* **MoveForward.cs**: Handles constant velocity for AI or background objects.

### 2. Farm Simulation Game
* **Interaction Logic**: Scripts designed for object triggers and environment boundaries.
* **DestroyOutOfBounds.cs**: Optimized memory management by removing objects that leave the play area.

## 🛠️ Technical Skills
* **C# Scripting**: Component-based architecture for game object behavior.
* **Game Engine**: Unity 3D with HDRP configuration.
* **Optimization**: Efficient boundary detection and object lifecycle management.
